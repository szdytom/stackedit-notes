ARC121F Logical Operations on Tree

题目大意

给定一棵有 nnn 个点的树的 n−1n-1n−1 条边 ai↔bia_i\leftrightarrow b_iai​↔bi​，现在对于要给每一个点加上权值 wi∈{0,1}w_i\in\{0,1\}wi​∈{0,1}，每一条边加上操作 oi∈{AND,OR}o_i\in\{\mathtt{AND},\mathtt{OR}\}oi​∈{AND,OR}

每次操作选择一条边收缩两边的点，权值为两者操作的结果

对于所有 22n−12^{2n-1}22n−1 种方案，求出所有存在一种收缩顺序使得最终剩余的点权值为 111 的加权值和操作的方案数。答案对 998244353998244353998244353 取模。

约束

  • 保证输入的图是一棵树
  • 2⩽n⩽1052\leqslant n \leqslant 10^52⩽n⩽105
  • 1⩽ai,bi⩽n1\leqslant a_i,b_i\leqslant n1⩽ai​,bi​⩽n

输入格式

第一行一个整数 nnn，然后 n−1n-1n−1 对数 ai,bia_i,b_iai​,bi​。
